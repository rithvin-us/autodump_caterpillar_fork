Caterpillar Innovation Challenge 2026 — Problem Statement 4

Optimal Dump Packing

Autonomous Cat 793 trucks dump on irregular mining fields using rigid pre-defined spot points. Operator safety buffers and field-agnostic grids leave corners and edges unfilled — about 62% coverage. AutoDump combines geometric zone decomposition, a trained Q-learning agent, and a deadlock-free token protocol to lift coverage with zero new hardware.

How it works

A horizontal sweep line scans from bottom to top, clipping the polygon against each truck-width strip using a ray-casting test. Each strip becomes a zone with a unique token. O(n log n) in vertex count.

How it works
On submit, the in-browser pipeline runs sweep-line decomposition (sliver-merge + MultiPolygon support) and then a greedy circle-packer. The trained Q-table only fits the original polygon, so custom shapes use the greedy packer.
